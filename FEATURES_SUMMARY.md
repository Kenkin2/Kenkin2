# Features Summary

## Core areas

Kin2 Workforce currently presents itself around these core areas:

- shift scheduling
- attendance and clock-in workflows
- timesheets and approvals
- payroll-support workflows
- staff records and compliance tasks
- communications and admin operations
- AI-assisted workflow support
