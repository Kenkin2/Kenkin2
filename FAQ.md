# FAQ

## What is Kin2 Workforce?

Kin2 Workforce is an AI-powered workforce operations platform for UK shift-based SMEs.

## Who is it for?

Kin2 is currently presented for sectors such as security, care, cleaning, hospitality, and charities.

## Is the main code public?

No. The main product repository is private.

## What does this public repository do?

This public repository provides a public-facing overview of the product, including status, release notes, legal summary, security contact, and related product pages.
