# Founder Note

Kin2 Workforce is being built to make workforce operations clearer, more practical, and more manageable for shift-based organisations.

The goal is not just to add more software, but to reduce friction around:
- scheduling
- attendance
- timesheets
- staff coordination
- compliance-related workflows
- day-to-day operational visibility

This public repository is part of that effort. It exists to give a clearer public view of what Kin2 Workforce is, where it is going, and how to learn more about it.
