# Architecture Overview

Kin2 Workforce is a platform for UK shift-based businesses.

## Product layers

- Web application for users and admins
- Backend services for business workflows and integrations
- Database for workforce, shift, timesheet, and reporting records

## Main capabilities supported

- scheduling
- attendance
- timesheets
- compliance workflows
- communications
- payroll-support workflows
